# 사주팔자 오늘의 운세

태어난 시각을 바탕으로 사주(연·월·일·시주)를 계산하고, 오행 균형과 오늘의 일진에 맞춘 운세 문장을 제공하는 단일 페이지 앱입니다.

## 주요 기능

- 생년월일/출생시간 입력을 통한 사주 4주 계산
- 절기(입춘 기준) 기반 연주·월주 판정
- 오행(목·화·토·금·수) 균형 분석 및 오늘의 기운 안내
- 운세 톤(담백/따뜻/직설) 선택에 따른 문장 변형
- 절기 계산값 테스트(참고 시각과 분 단위 차이 표시)
- 라이트/다크 테마 토글

## 구성 파일

- `index.html`: 화면 구조와 폼/결과 영역 정의
- `style.css`: 테마, 카드 UI, 반응형 레이아웃 스타일
- `main.js`: 사주 계산 로직, 절기 계산, 운세 생성, DOM 렌더링

## 실행 방법

별도의 빌드 없이 정적 파일로 동작합니다. 로컬에서 실행하려면 간단한 정적 서버를 띄우면 됩니다.

```bash
# 예시: Python 정적 서버
python -m http.server 8000
```

이후 브라우저에서 `http://localhost:8000`에 접속합니다.

## 사용 방법

1. 이름(선택), 생년월일, 출생시간을 입력합니다.
2. 성별(선택)과 운세 톤을 선택합니다.
3. "오늘의 운세 보기"를 눌러 결과를 확인합니다.
4. 필요 시 "절기 시각 정확도 테스트"에서 절기 계산값을 확인합니다.
5. 우측 상단의 테마 버튼으로 다크 모드를 전환합니다.

## 참고 사항

- 절기(입춘 기준)를 반영해 연·월주를 계산하지만, 실제 명식과는 오차가 있을 수 있습니다.
- 본 결과는 참고용이며, 중요한 결정은 현실적 판단과 함께 진행하세요.
